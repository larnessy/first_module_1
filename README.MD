## Основные команды Git

_init_ — инициализирует новый локальный репозиторий Git.  
Нужно вызывать из корневой директории проекта.  
```sh
git init
```

_clone_ — клонирует существующий удалённый репозиторий  
в локальную директорию. Для использования нужна ссылка,  
которую можно взять со страницы удалённого репозитория  
на Github (ссылка заканчиваетс на .git).  
```sh
git clone <repository_url>
```

_add_ — добавляет изменения в индекс (staging area)  
для подготовки к коммиту. «Делаем снимок текущего состояния файла(ов)».  
```sh
git add file.txt  # Добавить конкретный файл
git add --all      # Добавить все измененные файлы
``` 

_commit_ — создает коммит с текущими изменениями из индекса.  
«Сохраняем ранее сделанный снимок состояния проекта».  
```sh
git commit -m "message"
```

_push_ — отправляет изменения из ветки локального репозитория в ветку удалённого.  
Находиться нужно в соответствующей ветке локального репозитория.  
```sh
git push -u origin main
```

_pull_ — загружает изменения из ветки удалённого репозитория в ветку локального.  
Находиться нужно в соответствующей ветке локального репозитория.  
```sh
git pull
```
